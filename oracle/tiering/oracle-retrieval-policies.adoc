= Retrieval policies
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../../media/

[.lead]
The tiering policies control which Oracle database blocks are tiered from the performance tier to the capacity tier. Retrieval policies control what happens when a block that has been tiered is read.
