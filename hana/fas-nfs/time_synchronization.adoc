= Time synchronization
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ./../media/

[.lead]
You must synchronize the time between the storage controllers and the SAP HANA database hosts.

To do so, set the same time server for all storage controllers and all SAP HANA hosts.
